## Usage in Mermaid Themes

The Trimble brand colors are pre-configured in the following theme files (in project root):

- `themes/trimble.json` - Light theme with white background
- `themes/trimble-dark.json` - Dark theme with Hero Gray background

### Color Mapping

| Mermaid Element | Light Theme | Dark Theme |
